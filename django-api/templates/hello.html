class based view

このテンプレートは、class based viewで実装。Djangoのgenericsを使用するため簡単に実装できる。

DjangoはMVT（model, view, template）で、ちと分かりづらいがViewはRailsで言うコントローラ的な役割を果たす。

流れ的には、Viewでクラスを用意、URLパターンを設定、実際のテンプレートを作成、みたいな感じ。

このHTMLは、path('helloworld2/', HelloWorldClass.as_view())、class HelloWorldClass(TemplateView):、hello.htmlによって実装されている。

コピペ用コマンド

docker-compose run --rm app python manage.py startapp helloworldapp

アプリを作ったら

プロジェクトのsettings.pyにあるINSTALLED_APPSにアプリ名を追加する。
